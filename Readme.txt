相关库:
cloudlib1:
Http请求相关库，使用retrofit+okhttp，rxjava做事件驱动

dbmodule:
DB以及DB对应model相关库，使用greendao，gradle执行task可以自动通过model生成dao

gpuimagelibray:
实现滤镜等效果的图片处理库

framework:
项目通用的一些基础工具库


app:
项目相关的主app库
